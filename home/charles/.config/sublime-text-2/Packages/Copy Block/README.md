Sublime CopyBlock
=================

Sublime Text 2 Plugin to cut and copy blocks of text.


Usage
=====

Two new commands are added: `"copy_block"` and `"cut_block"`.
By default, these are bound to `ctrl+shift+c` and `ctrl+shift+x`,
respectively.
